Итак, как все устроено

В ExampleClass.js лежит класс Example - предок всех примеров в песочнице

Мысленно создадим объект Example, у него есть холст (Canvas) и методы типа draw() для рисования на нем

Cам по себе этот объект существует в вакууме, чтобы его увидеть надо его куда-нибудь встроить
для этого у него есть метод mount(parent) в любой HTML элемент контейнер, собсна

На главной странице index.html есть контейнер div с id="easel"(aka мольберт) и менюшка nav

у nav есть метод nav.add(exampleObject); этот метод создает в меню кнопку при нажатии на которую срабатывает exampleObject.mount(easel) и холст монтируется в мольберт

Итак, чтобы создать пример нужно:

1. let ololo = new Example('имя примера которое будет в меню'); создать объект Example (или его наследника)

2. nav.add(ololo); добавить его в меню

Если ты создал этот объект в отдельном файле, то его надо еще подключить к основному index.html добавив
<script src="путь к файлу/мой пример.js"></script> к другим ссылкам

Кстати эти файлы добавляются в меню в порядке подключения, и при открытии страницы в мольберт монтируется первый
