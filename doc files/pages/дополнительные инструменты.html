Дополнительные инструменты

В файле Main.js определены всякие удобные штуки

keysPressed - множество с кодами нажатых в данный момент кнопок клавиатуры

MBPressed - множество нажатых в данный момент кнопок мыши
0 - LMB, 2 - RMB

cursor - объект с текущими координатами курсора cursor.x и cursor.y

Функция function rgba(r = 0, g = 0, b = 0, a = 255) - возвращает объект цвета который ждет функция заполнения этюда

Этюд

Это еще один холст (буквально наследник HTMLCanvasElement) который нужен чтобы сначала на нем рисовать какой-то кусок картины, а потом рисовать его на основном холсте

У него есть метод fillWith(fillFunc) который принимает функцию заполнения и закрашивает каждый пиксель этюда согласно этой функции

fillFunc(x, y) принимает координаты пикселя этюда и возвращает объект цвета { r:r, g:g, b:b, a:a}

У него есть контекст рисования на нем самом this.ctx

Нарисовать этюд на основном холсте можно так же как и любой другой объект Canvas
this.ctx.drawImage(this.etude, 0, 0);
