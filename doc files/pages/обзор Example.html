Шо есть в Example?

В нем есть холст this.canvas и контекст this.ctx

Все рисование происходит на контексте рисования this.ctx, но ширина и высота холста берется из this.canvas.width this.canvas.height

Это не я придумал, так canvas сделан

Холст автоматически растягивается чтобы занять весь экран, можно не парится

Еще у него есть методы

По умолчанию после init() происходит draw(), а после afterResize() происходит init()

Не обязательно рисовать только в draw(), это просто выделеный для этого метод
